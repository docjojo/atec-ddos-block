atec DDoS block 
(Distributed Denial of Service Attacks)

== Description ==

This plugins observes incoming traffic and blocks repetitive requests from related IP addresses, thereby preventing DDOS attacks on your website.
For GDPR compliancy, IP addresses are deleted every 24 hours.
"atec DDoS block" will also protect your admin login form from brute force attacks.

No configuration required.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or upload the zip within WordPress
2. Activate the plugin through the `Plugins` menu in WordPress
3. Click "atec DDoS block" link in admin menu bar.
